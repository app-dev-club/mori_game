# Google Play ストア掲載情報（日本語）

## アプリ名

もり

## 短い説明

オンラインで遊べる、早い者勝ちの計算系カードゲーム「もり」。

## 詳細説明

「もり」は、場のカードと手札を使って勝利を狙うオンライン対戦カードゲームです。

場と同じ数字やスートのカードを出しながら、条件がそろったら「もり」を宣言。相手の宣言には「もり返し」で応戦できます。山札からカードを引くタイミングや、早い者勝ちで割り込む判断が勝敗を分けます。

手札の数字を足し算、引き算、掛け算、割り算で組み合わせて場の数字を作る、少し頭を使うルールも特徴です。シンプルなカード操作で、友達やオンラインの参加者とテンポよく対戦できます。

主な特徴:
- オンラインルームで対戦
- 公開ルームと非公開ルームに対応
- 早い者勝ちの割り込みプレイ
- 「もり」「もり返し」による逆転要素
- Bot 参加に対応
- 試合ログ、ランキング、設定機能
- ルールとプライバシーポリシーをアプリ内で確認可能

短時間で遊べるカードゲームを探している方、数字を使った駆け引きが好きな方におすすめです。

## アプリカテゴリ

- 種別: ゲーム
- カテゴリ候補: カード

## スクリーンショット案

Google Play には最低 2 枚が必要です。推奨として、スマートフォン縦向き 1080 x 1920 px を 4 枚以上用意します。

1. ホーム / ルーム作成画面
   - ルーム作成、ランキング、ルールなどの入口が分かる画面
2. 対戦画面
   - 場札、山札、手札、参加者が見える基本プレイ画面
3. もり / もり返しの場面
   - このアプリ独自の勝負どころが伝わる画面
4. ランキングまたは試合ログ
   - 継続して遊べる要素が分かる画面

スクリーンショットの注意:
- 実際のアプリ画面を使う
- 通知や個人情報が映らないようにする
- 追加テキストを入れる場合は画面の 20% 以下に抑える
- 「無料」「今すぐ」などの宣伝文句は入れない

## アプリアイコン

候補ファイル:

```text
store_assets/play/app_icon.png
```

要件:
- 512 x 512 px
- 32-bit PNG
- 最大 1024 KB

## フィーチャーグラフィック

候補ファイル:

```text
store_assets/play/feature_graphic.png
```

要件:
- 1024 x 500 px
- JPEG または 24-bit PNG
- 透過なし

## プライバシーポリシー URL

```text
https://mori-game.web.app/privacy.html
```

デプロイ前の確認:

```powershell
flutter build web --release
npx -y firebase-tools@latest deploy --only hosting --project mori-game
```

- Play Console へ提出する AAB:

```text

```
